Powershell Scripting

Een eindopdracht voor scripting die meetelde als examen, een script die automatisch de installatie en configuratie van een Windows Server als domeincontroller uitvoert. Het is makkelijk te gebruiken doormiddel van een menu.

Resultaat: Goed werkende script die makkelijk te gebruiken is.

Wat doet het script automatisch bij opstarten?

  • Netwerkconfiguratie aanpassen volgens computer.settings.xml
  • Domain Controller installeren of lid maken van een bestaand domein.
  • Organizational Units aanmaken uit ou.txt
  • Security Groups aanmaken uit securitygroups.csv
  • Domeingebruikers aanmaken uit users.json
  • Users toevoegen aan groepen volgens users.json
  • Mappen en shares aanmaken uit mappen.txt en shares.csv
  • NTFS-rechten toekennen aan mappen/shares volgens rechten.csv
